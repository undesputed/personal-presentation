# Personal Presentation

An interactive slide deck telling Carrie A. Yu's career story — **"From Zero to Leader"** — from software developer to team leader at Hipe Japan. Bilingual (English + Japanese), self-contained, and built as a single HTML file.

## Contents

- **[index.html](index.html)** — The 17-slide presentation. A "sunrise" theme (dawn → sunrise → daylight) that opens with a hello greeting and a short self-introduction, then follows the journey from the Philippines through the climb into leadership and AI work.
- **[about-me.html](about-me.html)** — An 8-slide bilingual (EN + JP) self-introduction in the same sunrise visual language as the main deck — who I am, career, sports, games, anime, learning & family, thank you & contact.
- **[qa-cheatsheet.html](qa-cheatsheet.html)** — A quick reference / Q&A cheat sheet to support the talk.

## Running it

No build step or dependencies. Just open the file in any modern browser:

```
open index.html
```

Fonts load from Google Fonts, so an internet connection is needed for the intended look.

## Controls

- **Next / Previous:** `→` `←` `↑` `↓`, `Space`, `Enter`, `Page Up/Down`, or the on-screen arrows
- **Jump to slide:** click any dot on the progress trail
- **First / last slide:** `Home` / `End`
- **Touch:** swipe left / right
- The current slide is saved in the URL (e.g. `#7`), so you can reload or share a link to a specific slide.

## Notes

- **Accessible:** honors `prefers-reduced-motion` and supports keyboard navigation.
- **Printable:** a print stylesheet lets you export the deck to PDF as a backup.
- **Responsive:** adapts to smaller screens and mobile.
